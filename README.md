## How to Run the Game (Windows)

1. Make sure you have a **JDK** installed on your device.
2. Unzip the file using WinRAR or 7-Zip.
3. Double-click **`play.bat`** — this will compile the game and launch it.
4. Start Playing.

## For MacOS, Arch, Debian, and Fedora Users:

1. Make sure you have a **JDK** installed on your device.
2. Open the directory in terminal.
3. Type **`chmod +x play.sh`** then Enter.
4. Type **`sed -i 's/\r$//' play.sh`** then Enter.
5. Type **`./play.sh`** then Enter.
6. Start Playing
